    1. Listas

    Las listas (list) son secuencias mutables que permiten almacenar cualquier tipo de dato y cuyo tama√±o puede crecer o disminuir din√°micamente. Los √≠ndices comienzan en 0 y se admiten √≠ndices negativos para contar desde el final (-1 es el √∫ltimo elemento). Adem√°s de acceder a un elemento concreto, el slicing (lista[inicio:fin:paso]) permite extraer sublistas sin bucles expl√≠citos.

    Operaciones principales

    • A√±adir elementos: append(x) agrega x al final; extend(iterable) concatena otra colecci√≥n; insert(i, x) inserta x antes del √≠ndice i.
    • Eliminar elementos: remove(x) borra la primera ocurrencia de x y lanza un error si no existe; pop([i]) extrae y devuelve el elemento de la posici√≥n i (por defecto, el √∫ltimo); clear() vac√≠a la lista.
    • Reordenar: sort() ordena la lista en su lugar (acepta argumentos key y reverse), mientras que sorted(lista) devuelve una nueva lista ordenada. reverse() invierte el orden de los elementos.
    • Buscar y contar: index(x) devuelve la posici√≥n de la primera aparici√≥n de x; count(x) devuelve cu√°ntas veces aparece x.
    • Copiar: lista.copy() o lista[:] devuelve una copia superficial.
    • Rebanado (slicing): lista[inicio:fin:paso] devuelve una sublista. Se pueden omitir inicio o fin, usar √≠ndices negativos para contar desde el final y especificar un paso distinto de 1 para saltar elementos o invertir el orden.

    2. Tuplas

    Las tuplas (tuple) son secuencias inmutables: una vez creadas, no se pueden a√±adir ni eliminar elementos. Son √∫tiles para agrupar datos heterog√©neos que no deben cambiar (coordenadas, fechas, etc.) y pueden utilizarse como claves en diccionarios al ser objetos hashables.

    Python permite desempaquetar una tupla en m√∫ltiples variables y convertirla a lista si se necesita modificarla.

    3. Conjuntos (set)

    Un conjunto (set) es una colecci√≥n no ordenada y sin elementos duplicados. Se utiliza para eliminar duplicados, comprobar pertenencia r√°pidamente y realizar operaciones de teor√≠a de conjuntos. Para crear un conjunto vac√≠o se usa set(), ya que {} crea un diccionario.

    Operaciones principales

    • Agregar o quitar: add(x) inserta x; remove(x) elimina x y lanza error si no existe; discard(x) elimina x si existe; pop() extrae un elemento arbitrario; clear() vac√≠a el conjunto.

    • Uni√≥n (| o union()): combina elementos de ambos conjuntos.

    • Intersecci√≥n (& o intersection()): elementos comunes a ambos conjuntos.

    • Diferencia (- o difference()): elementos presentes en un conjunto pero no en otro.

    • Diferencia sim√©trica (^ o symmetric_difference()): elementos en uno u otro pero no en ambos.

    4. Diccionarios

    Un diccionario (dict) almacena pares clave‚Äëvalor. Las claves deben ser √∫nicas e inmutables y se accede a los valores mediante la clave. Los diccionarios son ideales para asociar informaci√≥n y se utilizan frecuentemente en Python (equivalen a los maps en Java o a los arrays asociativos de PHP).

    Operaciones principales

    • Crear y modificar: se define con llaves y pares clave: valor. Se a√±ade o actualiza con dict[clave] = valor.

    • Eliminar: del dict[clave] elimina el par; pop(clave) extrae y devuelve el valor; clear() borra todo el diccionario.

    • Recorrer: keys() devuelve las claves; values() devuelve los valores; items() devuelve tuplas (clave, valor) para iterar.

    • Obtener valor seguro: get(clave, predeterminado) evita errores cuando la clave no existe.

    • Ordenar: sorted(diccionario) devuelve una lista de claves ordenadas.

    • Comprensiones de diccionario: permiten construir diccionarios de forma concisa.

    5. Comprensiones

    Las comprensiones permiten construir colecciones de manera compacta y legible.

    • List comprehensions: [expr for elem in iterable if condicion]. Permiten filtrar y transformar elementos.

    • Set comprehensions: {expr for elem in iterable} construyen un conjunto.

    • Dict comprehensions: {clave_expr: valor_expr for elem in iterable} construyen un diccionario.

    # Lista con los cuadrados de los n√∫meros impares del 0 al 10

# Conjunto de vocales √∫nicas en una frase

# Diccionario que mapea n√∫meros pares a su cubo si el cubo > 20
cubos = {x: x**3 for x in range(6) if x % 2 == 0 and x**3 > 20}

print(impares_cuadrados)
print(vocales)
print(cubos)
