    Estadísticos básicos en pandas

    1. Teoría

    Los estadísticos básicos permiten obtener una visión rápida del comportamiento de una variable numérica o categórica.
    En pandas, df.describe() resume columnas numéricas o categóricas y métodos como mean, median, std, quantile o value_counts complementan el análisis.

    2. Ejemplo sintético

    import pandas as pd
import numpy as np

np.random.seed(0)
n = 50

data = {
    "order_id": range(1, n + 1),
    "units": np.random.randint(1, 10, size=n),
    "unit_price": np.random.uniform(5.0, 20.0, size=n),
    "store": np.random.choice(["A", "B", "C"], size=n)
}

df = pd.DataFrame(data)
df["total"] = df["units"] * df["unit_price"]

print(df.describe())
print(df.describe(include="all"))

print("Media total:", df["total"].mean())
print("Percentil 25:", df["total"].quantile(0.25))
print("Frecuencia por tienda:")
print(df["store"].value_counts())

    3. Diagrama Mermaid

    flowchart TD A[Dataset cargado] --> B[Verificar tipos y limpiar] B --> C[Aplicar df.describe()] C --> D[Calcular mean, median, std, quantile] D --> E[Analizar categóricas con value_counts] E --> F[Interpretar tendencias y dispersión]

    4. Aplicación al dataset NYC.csv

    import pandas as pd

df = pd.read_csv("/mnt/data/NYC.csv")

cols = [
    "NUM_HERIDOS", "NUM_FALLECIDOS",
    "HERIDOS_PEATON", "HERIDOS_CICLISTA", "HERIDOS_CONDUCTOR"
]

for c in cols:
    df[c] = pd.to_numeric(df[c], errors="coerce")

print(df[cols].describe())
print(df[cols].mean())
print(df[cols].quantile([0.25, 0.75]))
print(df["DISTRITO"].value_counts())

    1. Crear columna GRAVEDAD y calcular media, mediana y std.

    3. Ejecutar describe(include="all") y analizar 3 columnas.
    4. Calcular media de heridos por distrito con groupby.
    5. Calcular percentiles 5, 50 y 95 para analizar simetría.

    6. Conclusión

    Los estadísticos básicos son esenciales para comprender tendencias, dispersión y distribución.
    Combinados con limpieza, tipificación y categorización, forman la base del análisis exploratorio de datos.
