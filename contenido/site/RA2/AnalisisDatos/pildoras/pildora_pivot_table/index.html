    Píldora formativa: Operaciones estructurales en DataFrames (pandas)

    1. Teoría

    En pandas, muchas tareas de análisis requieren crear nuevas columnas, reorganizar los datos o desplazarlos en el tiempo. Las funciones más representativas de este grupo son:

    assign

    Permite añadir columnas derivadas devolviendo un nuevo DataFrame con las columnas añadidas o modificadas, sin alterar el original si no se re-asigna la variable. Es especialmente útil para encadenar operaciones de forma clara.

    pivot_table

    Construye tablas dinámicas: reorganiza los datos usando filas, columnas, valores y una función de agregación (media, suma, conteo, etc.). Es ideal para resúmenes multidimensionales (por ejemplo, accidentes por distrito y por mes).

    shift

    Desplaza valores hacia arriba/abajo un número de periodos. Muy utilizado en series temporales, especialmente para crear lags (valor del día anterior, semana anterior, etc.) y calcular variaciones.

    last

    Aplicado después de un groupby, devuelve el último valor de cada grupo. Sirve para quedarnos con el valor más reciente de una serie diaria, mensual o para cada categoría.

    info

    No transforma los datos, pero es clave para inspeccionar estructura, tipos y valores nulos. Suele usarse antes de aplicar assign, pivot_table o shift para verificar la calidad de los datos.

    Estas funciones se combinan frecuentemente en flujos como:
    inspección (info) → creación de columnas (assign) → reorganización (pivot_table) → análisis temporal (shift, last).

    2. Ejemplo sintético (con gráficos)

    Imaginemos una pequeña tabla de ventas diarias por categoría. Queremos:

    • Crear una columna normalizada (assign).
    • Calcular el valor del día anterior (shift).
    • Resumir la información en una tabla dinámica (pivot_table).
    • Graficar tanto la serie como el lag.
    import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

df = pd.DataFrame({
    "fecha": pd.date_range("2024-01-01", periods=6, freq="D"),
    "categoria": ["A","A","B","B","B","A"],
    "valor": [10, 12, 5, 7, 9, 11]
})

# 1. info: inspección
df.info()

# 2. assign: columna derivada (normalización)
df = df.assign(valor_norm=df["valor"] / df["valor"].max())

# 3. shift: lag de 1 día
df["valor_lag1"] = df["valor"].shift(1)

# 4. pivot_table: media de valor por categoría y mes
df["mes"] = df["fecha"].dt.month
tabla = df.pivot_table(values="valor", index="categoria", columns="mes", aggfunc="mean")

print(df)
print(tabla)

# --- Gráficos automáticos ---

# A) Serie de valor y lag en el tiempo
plt.figure()
plt.plot(df["fecha"], df["valor"], marker="o", label="valor")
plt.plot(df["fecha"], df["valor_lag1"], marker="x", linestyle="--", label="valor_lag1")
plt.title("Valor y valor_lag1 en el tiempo")
plt.xlabel("Fecha")
plt.ylabel("Valor")
plt.legend()
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()

# B) Gráfico de barras de la tabla dinámica (media por categoría y mes)
plt.figure()
tabla.plot(kind="bar")
plt.title("Valor medio por categoría y mes")
plt.xlabel("Categoría")
plt.ylabel("Valor medio")
plt.tight_layout()
plt.show()

    En este ejemplo:

    • assign añade una columna normalizada sin romper el flujo encadenado.
    • shift nos permite comparar el valor actual con el del día anterior.
    • pivot_table resume el valor medio por categoría y mes.
    • Los gráficos facilitan ver la diferencia entre la serie original y su lag, y comparar categorías.

    3. Diagrama Mermaid

    flowchart TD A[df.info()] --> B[assign: crear columnas] B --> C[shift: generar lags temporales] C --> D[pivot_table: reorganizar datos] D --> E[last: obtener últimos valores por grupo]

    4. Aplicación al dataset NYC.csv (con gráficos)

    Trabajaremos con el dataset de accidentes de tráfico en NYC (NYC.csv). El objetivo es:

    • Inspeccionar la estructura (info).
    • Crear columnas temporales (assign).
    • Estudiar accidentes diarios y su lag (shift).
    • Resumir heridos por distrito y mes (pivot_table).
    • Obtener el último valor registrado por distrito (last).
    • Visualizar los resultados con gráficos automáticos.
    import pandas as pd
import matplotlib.pyplot as plt

# 1. Cargar el dataset
df = pd.read_csv("/mnt/data/NYC.csv")

# 2. Inspección de estructura
df.info()

# 3. Conversión de fecha y columnas derivadas
df["FECHA_ACCIDENTE"] = pd.to_datetime(df["FECHA_ACCIDENTE"], errors="coerce")

df = df.assign(
    anio=df["FECHA_ACCIDENTE"].dt.year,
    mes=df["FECHA_ACCIDENTE"].dt.month
)

# 4. Accidentes por día
acc_diarios = df.groupby("FECHA_ACCIDENTE").size().rename("num_acc")
acc_df = acc_diarios.reset_index()

# 5. Lag de 1 día con shift
acc_df["num_acc_lag1"] = acc_df["num_acc"].shift(1)

print(acc_df.head())

# --- Gráfico 1: accidentes diarios y lag ---
plt.figure()
plt.plot(acc_df["FECHA_ACCIDENTE"], acc_df["num_acc"], label="num_acc")
plt.plot(acc_df["FECHA_ACCIDENTE"], acc_df["num_acc_lag1"], linestyle="--", label="num_acc_lag1")
plt.title("Accidentes diarios y lag de 1 día")
plt.xlabel("Fecha")
plt.ylabel("Número de accidentes")
plt.legend()
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()

# 6. Tabla dinámica: heridos por distrito y mes
tabla = df.pivot_table(
    values="NUMERO_PERSONAS_HERIDAS",
    index="DISTRITO",
    columns="mes",
    aggfunc="sum"
)

print(tabla.head())

# --- Gráfico 2: calor aproximado por distrito (barras apiladas por mes) ---
plt.figure()
tabla.plot(kind="bar", stacked=True)
plt.title("Personas heridas por distrito y mes")
plt.xlabel("Distrito")
plt.ylabel("Total de personas heridas")
plt.tight_layout()
plt.show()

# 7. Último valor registrado de heridos por distrito (last)
ultimos = (
    df.sort_values("FECHA_ACCIDENTE")
      .groupby("DISTRITO")["NUMERO_PERSONAS_HERIDAS"]
      .last()
)

print(ultimos.head())

# --- Gráfico 3: último valor de heridos por distrito ---
plt.figure()
ultimos.sort_values(ascending=False).plot(kind="bar")
plt.title("Último valor registrado de heridos por distrito")
plt.xlabel("Distrito")
plt.ylabel("Heridos")
plt.tight_layout()
plt.show()

    En este flujo:

    • df.info() te permite ver rápidamente qué columnas están disponibles, sus tipos y cuántos nulos tienen.
    • assign añade columnas temporales (anio, mes) sin romper la cadena de operaciones.
    • shift genera un lag de accidentes diarios, útil para analizar subidas/bajadas de un día a otro.
    • pivot_table resume heridos por distrito y mes, y la representación en barras apiladas ofrece una idea visual del peso relativo de cada mes en cada distrito.
    • last te da el último registro de heridos por distrito (por ejemplo, el último día con datos para cada uno), y el gráfico de barras facilita comparar rápidamente distritos.

    6. Conclusión

    Las funciones assign, pivot_table, shift, last e info conforman un conjunto muy útil para enriquecer y reorganizar DataFrames:

    • info te dice en qué estado están los datos.
    • assign te permite añadir columnas derivadas con un estilo declarativo y encadenable.
    • shift facilita trabajar con series temporales y comparar con periodos anteriores.
    • pivot_table transforma datos “largos” en tablas de resumen con múltiples dimensiones.
    • last te ayuda a quedarte con el valor más reciente por grupo.

    Combinadas con visualizaciones automáticas en matplotlib, estas herramientas permiten pasar de datos crudos a resúmenes visuales que cuentan la historia de los datos de forma clara y accionable.
