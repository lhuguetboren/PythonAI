    Variables categóricas con astype('category') aplicadas al dataset de accidentes de NYC

    Teoría

    Una variable categórica es aquella que toma un número limitado y fijo de valores, que representan categorías o niveles. Ejemplos típicos son el país, la ciudad, la talla de un producto ("S", "M", "L") o el método de pago. En pandas estas variables se representan con el tipo category. La documentación oficial indica que un objeto Categorical mantiene internamente un vector de categorías y un vector de códigos enteros que apuntan a dichas categorías. Esta estructura reduce el uso de memoria y acelera operaciones cuando hay muchas repeticiones. Además, es posible definir un orden lógico distinto del orden alfabético para las categorías.

    Ventajas principales

    • Ahorro de memoria: las cadenas repetidas se reemplazan por códigos enteros.
    • Eficiencia en agrupaciones y comparaciones: los códigos numéricos son más rápidos de comparar que las cadenas.
    • Ordenación lógica: permite ordenar categorías según un criterio definido por el usuario (por ejemplo, tamaños S < M < L).
    • Señalización para otras bibliotecas: algunas librerías de análisis o gráficos interpretan las columnas category de forma diferente.

    Ejemplo sintético

    El archivo original proporcionaba un ejemplo completo para trabajar con variables categóricas. A continuación se resume la esencia de ese ejemplo: se crea un DataFrame con columnas de texto y se utilizan astype('category') para transformarlas a categorías. Se revisan los códigos internos, se define un orden lógico y se realiza una agrupación.

    import pandas as pd

# 1. Datos sintéticos
data = {
    "customer_id": [1, 2, 3, 4, 5, 6],
    "country": ["ES", "ES", "FR", "DE", "ES", "FR"],
    "product_size": ["M", "L", "S", "M", "M", "L"],
    "payment_method": ["Card", "Cash", "Card", "Card", "App", "Cash"]
}

df = pd.DataFrame(data)

# 2. Conversión a categorías
df["country"] = df["country"].astype("category")
df["product_size"] = df["product_size"].astype("category")
df["payment_method"] = df["payment_method"].astype("category")

# 3. Inspeccionar categorías y códigos
print(df["country"].cat.categories, df["country"].cat.codes.tolist())

# 4. Definir orden lógico en product_size
from pandas.api.types import CategoricalDtype
size_order = CategoricalDtype(categories=["S", "M", "L"], ordered=True)
df["product_size"] = df["product_size"].astype(size_order)

# 5. Groupby con variable categórica
df["units"] = [3,1,4,2,5,2]
print(df.groupby("country")["units"].sum())

    Este ejemplo muestra cómo astype('category') permite transformar varias columnas a categorías y aprovechar su estructura para ordenar o agrupar datos.

    Aplicación al dataset de accidentes de NYC

    El conjunto de datos NYC.csv contiene registros de accidentes de tráfico en Nueva York. Muchas de sus columnas son cadenas (object) con valores repetidos, ideales para convertirlas a variables categóricas. A continuación se muestra cómo utilizar astype('category') para reducir el tamaño en memoria y analizar los valores más frecuentes en una columna categórica.

    import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Cargar el dataset
df = pd.read_csv('data/NYC.csv')

# Memoria antes de convertir
mem_before = df.memory_usage(deep=True).sum() / 1024**2

# Seleccionar columnas con pocos valores únicos y convertir a category
categorical_cols = ['DISTRITO', 'TIPO_VEHICULO_1', 'FACTOR_CONTRIBUYENTE_VEHICULO_1']
for col in categorical_cols:
    df[col] = df[col].astype('category')

# Memoria después de convertir
mem_after = df.memory_usage(deep=True).sum() / 1024**2
print(f"Memoria antes: {mem_before:.2f} MB; después: {mem_after:.2f} MB")

# Calcular los 10 tipos de vehículo más frecuentes
top_vehicle_counts = df['TIPO_VEHICULO_1'].value_counts().head(10)
print(top_vehicle_counts)

# Visualizar los resultados
a = plt.figure(figsize=(10,5))
sns.barplot(x=top_vehicle_counts.index, y=top_vehicle_counts.values, palette='plasma')
plt.xticks(rotation=45, ha='right')
plt.title('Top 10 tipos de vehículos implicados en accidentes')
plt.xlabel('Tipo de vehículo')
plt.ylabel('Número de accidentes')
plt.tight_layout()

# Guardar imagen si se desea incluir en un informe
plt.savefig('top10_vehiculos.png', dpi=300)

    En este ejemplo, las columnas DISTRITO, TIPO_VEHICULO_1 y FACTOR_CONTRIBUYENTE_VEHICULO_1 se convierten a category utilizando astype. La memoria total consumida por el DataFrame se reduce de aproximadamente 71,39 MB a 57,12 MB gracias a la codificación interna de las categorías. Además, el código utiliza value_counts() para obtener los diez tipos de vehículo más frecuentes y los grafica con seaborn; el uso de astype('category') simplifica este análisis al indicar explícitamente que se trata de una variable categórica.

    Actividades propuestas

    1. Comparar ahorro de memoria: Identifique otras columnas de NYC.csv con pocos valores distintos (por ejemplo, NOMBRE_CALLE_PRINCIPAL o DISTRITO) y conviértalas a category. Calcule la memoria utilizada antes y después de la conversión.
    2. Análisis de factores contribuyentes: Tras convertir FACTOR_CONTRIBUYENTE_VEHICULO_1 a categoría, utilice value_counts() para mostrar los factores más comunes en los accidentes. Visualice los cinco primeros con un gráfico de barras. Recuerde incluir el código para la gráfica.
    3. Ordenar categorías temporales: Agrupe las horas de accidente (HORA_ACCIDENTE) en intervalos (mañana, tarde, noche) usando pd.cut o una función personalizada, convierta la nueva columna a categoría ordenada y analice en qué franja horaria se producen más accidentes.
    4. Combinación de categorías: Cree una nueva variable categórica que combine el tipo de vehículo y el distrito (por ejemplo, "Sedan-Brooklyn"). Conviértala a category y utilice value_counts() para encontrar las combinaciones más comunes.

    Conclusión

    El tipo category de pandas es fundamental para representar variables con un número limitado de valores. Según la documentación, las variables categóricas almacenan las categorías y los códigos internos de cada valor, lo que reduce considerablemente el uso de memoria y mejora la eficiencia en operaciones como agrupaciones. En el dataset de accidentes de Nueva York, convertir columnas con valores repetidos a category mediante astype reduce el tamaño en memoria y facilita el análisis de los tipos de vehículos o factores que intervienen en los accidentes. Estas técnicas permiten optimizar tanto el rendimiento como la claridad del código cuando se trabaja con datos categóricos.
